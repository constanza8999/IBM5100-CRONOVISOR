◈ About the IBM 5100

Released in September 1975, the IBM 5100 was one of the world's first portable computers. Weighing approximately 55 pounds (25 kg), it featured a built-in 5-inch CRT display, a tape cartridge drive, and came with both BASIC and APL programming language interpreters built into ROM.

At the heart of the IBM 5100 was the PALM (Put All Logic in Microcode) processor — a 16-bit CPU implemented entirely in microcode. The machine could be equipped with 16 KB to 64 KB of memory and cost between $9,000 and $24,000 (roughly $50,000–$130,000 in 2025 dollars).

Despite its high price, the IBM 5100 was a revolutionary machine — it was the first computer to bundle a high-level language interpreter in ROM, and its portable design (if you were strong enough to carry it!) foreshadowed the laptop revolution. It also holds a special place in pop culture through the visual novel/anime Steins;Gate, where the IBM 5100's unique architecture plays a key role in the plot.

◈ How to Use

The emulator starts with an authentic boot sequence. Once it shows READY, type a command and press Enter.

BASIC Mode (Default)

Enter numbered lines to create programs:

10 PRINT "HELLO WORLD"
20 FOR I = 1 TO 5
30 PRINT I
40 NEXT I
RUN

APL Mode

Switch with MODE APL. Use special APL symbols:

2+2         →  4
3×4         →  12
10÷2        →  5
(2+3)×(10-4) →  30
