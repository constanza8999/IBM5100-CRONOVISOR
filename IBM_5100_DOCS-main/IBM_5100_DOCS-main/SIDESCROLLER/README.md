
The project here is PALM machine code written in 2025 that
generates a "demoscene-style" horizontal side-scroller that
runs on a IBM 5100.

A complete walk-thru of the code is in a PowerPoint/PDF presentation.
Excel was used to help encode the image-glyphs.
Other documentation covers aspects of the PALM instruction set, which
was originally introduced in 1972.

This code was assembled and presented in a working IBM 5100 at VCF Southwest 2025.

Various folks helped make this possible, read the backstory in 
the December 2025 issue of COMPUTE! Gazette!

A video of running this code is available on YouTube:
